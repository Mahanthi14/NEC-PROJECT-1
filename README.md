# AI-Driven Customer Intelligence Dashboard

An end-to-end AI-powered business intelligence platform designed to analyze customer behavior, identify valuable customer segments, predict churn risk, recommend products, monitor inventory levels, and generate actionable business insights through interactive dashboards and visualizations.
Live link : https://nec-project-1-3365.onrender.com
## Features

* Customer Segmentation using K-Means Clustering
* Churn Prediction using Machine Learning
* Purchase Value Prediction
* Product Recommendation System
* Inventory Stock Monitoring & Alerts
* Dynamic Customer Filtering
* Interactive Bar Charts, Pie Charts, and 3D Visualizations
* Old and New Customer Data Integration
* Automated PDF Report Generation
* Streamlit-Based Interactive Dashboard

## Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-Learn
* Plotly
* FPDF

## Business Impact

This dashboard helps businesses understand customer behavior, improve marketing strategies, reduce customer churn, optimize inventory management, and make data-driven decisions through AI-powered analytics.
